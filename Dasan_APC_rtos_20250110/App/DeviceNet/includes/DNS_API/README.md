# DNS_API Directory

This directory should contain the DeviceNet packet API header files.

## Files to Copy from DeviceNet SDK

Copy all DNS (DeviceNet Slave) API header files from the DeviceNet SDK to this directory, including:

- DNS packet handling headers
- DeviceNet protocol definitions
- Network configuration structures
- I/O data structures

## Source Location

These files should be copied from the DeviceNet SDK's API include directory.

## Integration Notes

These headers provide the low-level DeviceNet protocol interface used by the application layer.
